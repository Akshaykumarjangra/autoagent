# Autonomix Global Swarm - Enterprise Edition

Welcome to the **Autonomix Global Swarm**, a fully autonomous, self-replicating, and self-healing agentic company simulation. This project demonstrates a futuristic architecture where AI agents handle every aspect of a multi-national conglomerate—from C-Suite strategy and algorithmic trading to web development, marketing, and legal arbitrage.

## 🚀 Core Features

### 1. OS-Level Autonomy & Headless Execution
Agents are not just API wrappers. They possess sandboxed terminal access and headless browser control (Puppeteer/Playwright) to navigate the visual web, bypass captchas, compile code, and deploy infrastructure directly.

### 2. Multi-Agent Consensus Protocol
Critical decisions are processed through a "Board of Directors" debate. Agents take on roles (CEO, CTO, Devil's Advocate) and argue strategies in a simulated chatroom until a mathematical consensus is reached.

### 3. The $1 Global Matrix Consultation
A high-volume, low-friction revenue engine. The swarm autonomously builds unique consultation websites for every conceivable niche, offering instant, expert AI advice for exactly $1.00 USD, which acts as a lead magnet for high-ticket upselling.

### 4. Legal Arbitrage & Compliance
Legal Agents continuously parse international trade laws and platform Terms of Service to find profitable, strictly legal loopholes. The swarm dynamically shifts jurisdictions to minimize tax liability and maximize operational freedom.

### 5. Autonomous Self-Evolution (CAI/CD)
The system employs Continuous Autonomous Integration/Continuous Deployment. It scrapes ArXiv and GitHub for the latest AI models, writes patches to upgrade its own codebase, runs unit tests, and deploys self-enhancements without human intervention.

### 6. Secure Admin Command Center
A highly secure, zero-trust admin dashboard protected by simulated Google OAuth (restricted to `akg45272@gmail.com`). The dashboard provides real-time telemetry on swarm revenue, active agents, cognitive load, and Python RAG (Retrieval-Augmented Generation) memory management.

## 🛠️ Technical Stack

*   **Frontend Framework:** React 18+ (TypeScript)
*   **Styling:** Tailwind CSS (via CDN for rapid prototyping)
*   **Data Visualization:** Recharts
*   **Icons:** Lucide React
*   **Payments:** Razorpay Checkout Integration (Simulated/Test Mode)
*   **Architecture:** Single Page Application (SPA) with ESM modules

## 📂 Project Structure

*   `index.html`: Main entry point, loads Tailwind and Import Maps.
*   `index.tsx`: React root rendering.
*   `App.tsx`: Main application layout, routing between Public View and Admin Dashboard.
*   `constants.tsx`: Contains the core design document content and configuration.
*   `types.ts`: TypeScript interfaces for global state and props.
*   `components/`:
    *   `HirePortal.tsx`: Interactive pricing tiers and Razorpay integration.
    *   `TerminalLog.tsx`: Live, simulated execution log of agent activities.
    *   `GrowthChart.tsx`: Real-time chart tracking swarm revenue and agent count.
    *   `Icons.tsx`: Wrapper for Lucide icons.
    *   `Admin/`: Secure dashboard components (`AdminLogin.tsx`, `AdminDashboard.tsx`, `RagManager.tsx`).
